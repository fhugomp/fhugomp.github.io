## 🚀 Funcionalidades da Interface

Este portfólio foi construído sem frameworks, focado em performance e boas práticas de UI/UX:

- **Sistema de Temas (Claro/Escuro):** Transição fluida de cores utilizando variáveis nativas do CSS (`:root`) e memória via `localStorage`.
- **Design Responsivo (Mobile First):** Layout adaptável para dispositivos móveis, tablets e desktops com o uso de _Media Queries_, CSS Grid e Flexbox.
- **Animações e Micro-interações:** Efeito de _fade-in_ ativado via scroll, elevação de cards dinâmicas (_hover_ com box-shadow) e animação de digitação (máquina de escrever).
- **Menu de Navegação Dinâmico:** Cabeçalho _sticky_ fixo com botão inteligente e menu _overlay_ para telas menores.
